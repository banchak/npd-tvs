ข้อ 3. กำหนดเวลาของสัญญา

3.1 คู่สัญญาทั้งสองฝ่ายตกลงกันว่า สัญญาจ้างบริการฉบับนี้มีกำหนดระยะเวลา ตั้งแต่วันที่ {{ resource.info.rent_date | moment:'thaitext' }} ถึงวันที่ {{ resource.info.rent_until | moment:'thaitext' }} และผู้รับบริการเริ่มชำระค่าบริการ ตั้งแต่วันที่ {{ resource.info.pay_date | moment:'thaitext' }} ถึงวันที่ {{ resource.info.pay_until | moment:'thaitext' }}

ผู้รับบริการไม่ต้องชำระค่าบริการ ในระยะเวลาที่ตกแต่งสถานที่เช่า นับตั้งแต่วันที่ {{ resource.info.decorate_date | moment:'thaitext' }} ถึงวันที่ {{ resource.info.decorate_until | moment:'thaitext' }}

คู่สัญญาตกลงกันว่า สัญญาจ้างบริการสาธารณูปโภคฉบับนี้ มีกำหนดเวลาเท่ากันกับสัญญาเช่าสถานที่ในศูนย์การค้า ธัญญะ ช็อปปิ้ง พาร์ค และสัญญาจ้างบริการ เมื่อสัญญาเช่า และ/หรือ สัญญาจ้างบริการ สิ้นสุดลง ให้สัญญาจ้างบริการสาธารณูปโภคฉบับนี้สิ้นสุดลงด้วย
