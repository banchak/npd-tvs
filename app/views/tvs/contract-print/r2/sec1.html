ข้อ 1. บริเวณสถานที่ที่เช่า

ผู้ให้เช่าตกลงให้เช่า ผู้เช่าตกลงเช่าพื้นที่ในโครงการธัญญะ ช็อปปิ้ง พาร์ค ซึ่งตั้งอยู่บนโฉนดที่ดินเลขที่ 183772, 184699, 196017, 206094, 31837-8, 77377-8, 77380-3, 197411-2, 192443-4, 233275, 225353, 2526, 74201 และ 6689 รวม 21 โฉนด ถนน ศรีนครินทร์ แขวง สวนหลวง (คลองประเวศฝั่งเหนือ)เขต พระโขนง กรุงเทพมหานคร {{ resource.display.area }} เนื้อที่ประมาณ {{ resource.info.area_xsize && '*' }}{{ resource.info.area_size }} ตารางเมตร บริเวณพื้นที่เช่าปรากฏตามแผนผังแนบท้ายสัญญาฉบับนี้ และให้ถือเป็นส่วนหนึ่งของสัญญาเช่าฉบับนี้ (ซึ่งต่อไปในสัญญานี้จะเรียกว่า "สถานที่เช่า")

ผู้ให้เช่าให้เช่าเฉพาะพื้นที่ภายในบริเวณสถานที่เช่าเท่านั้น ผู้เช่าไม่มีสิทธิภายนอกพื้นที่สถานที่เช่า ผู้ให้เช่าทรงไว้ซึ่งสิทธิแต่เพียงผู้เดียวในการใช้ประโยชน์ภายนอกสถานที่เช่า

* หมายเหตุ พื้นที่รังวัดแล้ว {{ (resource.info.area_xsize<0 && 'ลดลง') || 'เพิ่มขึ้น' }} จากสัญญาจอง {{ resource.info.area_xsize }} ตารางเมตร
